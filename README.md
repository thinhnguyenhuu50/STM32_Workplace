# Danh sách bài dạy
**Bài 1 - GPIO, Delay**
* Giới thiệu về GPIO.
* Giới thiệu về delay.

**Bài 2 - Timer**
* Giới thiệu về Timer.
* Sử dụng 1 Timer chung để định thời cho tất cả các tác vụ khác.

**Bài 3 - LCD, Button**
* Giới thiệu về thư viện Nút nhấn và Màn hình LCD.

* Giới thiệu về Máy trạng thái - (finite-state machine FSM).
